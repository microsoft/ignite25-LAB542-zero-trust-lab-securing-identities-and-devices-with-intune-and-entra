### Lab Description

This repository contains the complete content for the **Zero Trust Lab: Securing Identities and Devices with Intune and Entra** hands-on lab. This lab provides practical experience implementing Zero Trust security principles using Microsoft Intune and Microsoft Entra ID to secure identities and devices in modern enterprise environments.

### ðŸ§  Learning Outcomes

By the end of this lab, learners will be able to:

- Implement Zero Trust security principles using Microsoft Intune and Entra ID
- Configure device compliance policies and conditional access rules
- Set up identity protection and risk-based authentication
- Manage device enrollment and security baselines
- Monitor and respond to security threats in a Zero Trust environment
